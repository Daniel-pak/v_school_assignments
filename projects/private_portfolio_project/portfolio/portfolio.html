Full Applications

The Moves - The event finder

A full stack application that allows people to find events - by name and by keyword category types - as well as create them if they are signed up as a business user. As a user, you can save events, look up more information on said event through links and a small blurb, and search for the location on a map. The application uses user authentication to allow sign in and sign out, Google Maps API, and Materialize styling to bring a real application feeling.

Foodie - The Next Big Recipe Finder

Foodie is a full CRUD application that allows users to search for recipes through ingredient, category, or by hot items of the day and save these recipes to look up for later. The application uses a RESTful API through angular http requests to allow my AngularJS client to display this information in a nicely bootstrapped website. The website was built with HTML5, CSS3, JavaScript, Angular, and Node.js (Express).

Exploration - Seeing Space and Earth through NASA's eyes

Exploration is a website that is built to inform it’s viewers on the activity of space. Through uses from NASA’s RESTful API, viewers can: learn from a daily article that is displayed with a picture, view our near-earth objects and it’s details, see a satellite image of their location (using their IP address), and see pictures from the rover - Curiosity - on Mars. The website was built using AngularJS, HTML5, CSS3, and Bootstrap.
